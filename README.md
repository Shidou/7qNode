# 7qNode
前后端分离框架

## 使用说明
> 处理JSON
> 成功的时候调用res.success();接受两个参数,第一个数数据,第二个是message(可不填);
> 失败的时候调用res.fail();接受两个参数,第一个参数是错误信息Error,第二个是错误描述,可以不填;

## 关于日志
移除logjs4,使用alinode和[pm2管理日志](https://wohugb.gitbooks.io/pm2/content/features/log.html)
